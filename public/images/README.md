# Site images

Upload your images into the folders below. Each folder has a **README** listing the exact filenames to use.

| Folder | Used on |
|--------|---------|
| `homepage_image/` | Homepage hero carousel |
| `class_image/` | Classes (homepage, classes page, modals) |
| `instructor_image/` | Instructors page |
| `about_us_image/` | About Us page (hero, team, mission, vision, goals, history) |

**Formats:** jpg, jpeg, png, webp

Images appear automatically once the file is saved with the correct name. Until then, the site shows a placeholder.
